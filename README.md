# Employee Management System

A real-time employee management system built with **Laravel** and **Vue.js**. This project includes a responsive frontend with live updates, allowing users to view and manage employee records dynamically. The app showcases essential features like pagination, real-time notifications, and customizable search filters.

---

## Features

- **Real-Time Updates**: New employees are instantly displayed in the UI using Pusher for live broadcasting.
- **Toast Notifications**: Users are notified of new employee additions via toast notifications, enhancing feedback and UX.
- **Dynamic Pagination**: Allows customization of results per page, with limited pagination links for easier navigation.
- **Responsive Design**: Works seamlessly across desktops and mobile devices.
- **Search and Filter**: Real-time search functionality to quickly find employees by name or other attributes.

---

## Technologies Used

- **Laravel** (Backend API and Broadcasting)
- **Vue.js** (Frontend Framework)
- **Pusher** (Real-Time Broadcasting)
- **Vue Toastification** (Toast Notifications)
- **Bootstrap** (UI Styling)
